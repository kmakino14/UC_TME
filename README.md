# UC_TME
This repository contains custom scripts and analysis pipelines for single-cell multiomics studies of urothelial carcinoma, including DOGMA-seq, spatial multiome, and MAESTER data. The code is organized by project and analysis step, including integration, visualization, and downstream analysis.

## Reproducibility & Data Policy
Processed datasets generated and analyzed in this study, including annotated AnnData (.h5ad) objects, cell-level metadata, spatial transcriptomics data, MAESTER-derived data, and embedding files, are available from Zenodo:
https://zenodo.org/records/21122266

Raw sequencing data are subject to controlled access and are available through appropriate repositories in accordance with ethical approval and data-sharing agreements and are deposited at the Japanese Genotype-phenotype Archive (JGA) under accession number JGAS000860.

Data are released for academic use, and users are asked to cite the corresponding Zenodo records and the associated publication when using these resources.
