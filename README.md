# UC_TME
This repository contains custom scripts and analysis pipelines for single-cell multiomics studies of urothelial carcinoma, including DOGMA-seq, spatial multiome, and MAESTER data. The code is organized by project and analysis step, including integration, visualization, and downstream analysis.

## Reproducibility & Data Policy
Keep large or licensed files out of the repository (.gitignore already handles this).
Publish only small metadata in data/list/ and barcode CSVs in data/TotalSeq/ unless all licenses explicitly allow redistribution.
The final Seurat objects generated in this project have been uploaded to Zenodo and can be accessed via the following DOIs:

TCR100-CITE: https://doi.org/10.5281/zenodo.17174369
TCR100-icCITE: https://doi.org/10.5281/zenodo.17174371
TCR100-ASAP: https://doi.org/10.5281/zenodo.17174337
These deposits ensure long-term accessibility and reproducibility of the processed datasets.
